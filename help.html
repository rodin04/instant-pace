User Input
The input for distance must be given in meters, and the input for time must always be in minutes! For example: 100 meters in 0.125 (not 12.5), or 5000 in 20.30 minutes. You can use either a comma ( , ) or a dot ( . ) for decimal numbers – both work fine. User input example
100m in 12,5 sek 5000m in 20,30min
